# draw
抽题目小工具
1. 页面展示题目列表和一个抽题按钮，同时抽取题目和答题序号（即第几位答）；
2. 用户点击抽题跳出弹框让其输入姓名
3. 输完姓名后点击保存后（不可为空）自动抽题，将姓名作为参数传到server
4. 若用户输入的姓名为配置文件中的admin用户则当其点击抽签时，默认抽到第5题，答题序号为10,将配置文件中 sequence seq=5,name置为admin用户的姓名,topic置为10；若用户输入的姓名不为admin用户，则其答题序号在sequence中随机抽取一个name为空的项(不能为seq=5)，将其name置为用户输入的姓名,topic 从topics列表里随机取一道;
5. 在用户页面上展示当前抽题的结果，即sequence里所有name不为空的项 用 序号: $seqNo,姓名:$name,题目:$topic 展示出来，每个用户占一行
6. 13位用户抽完后，保证sequence里所有name都不为空，topic也都不为空；且topics里的所有题目都被抽到过
7. 用现代的交互习惯和设计方式，界面简介美观
8. 添加一个/reset接口，当用户点击该接口时，将sequence里所有name置为空，topic也置为空；且将topics里的所有题目都设置为未被抽到过
9. 补充校验逻辑，相同用户名称视为同一个人，抽签时要先检查该用户是否已经抽过题，存在则以之前的抽题结果为准。
10. 补充场景，需要满足不同用户在不同设备上进行抽取，所有对sequence的修改应该在服务器进行并保证只有一份sequence 所有用户都对这份sequence进行修改和读取¡
